🔑 Gemini API 키 입력
AI 일정 생성을 위해 본인의 Gemini API 키가 필요해요.
키는 이 기기 메모리에만 저장되며 탭을 닫으면 자동 삭제돼요.
서버에는 전달만 되고 저장되지 않아요.
무료 키 발급: aistudio.google.com/api-keys
📖 사용 방법
1 도시, 날짜, 동행자, 분위기를 설정해주세요
2 가고 싶은 장소를 추가하세요 (방문 희망 시간도 지정 가능)
3 AI 여행 일정 생성하기 버튼을 클릭하세요
4 완성된 일정을 확인하고 필요하면 장소를 교체하거나 PDF로 저장하세요
